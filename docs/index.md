# json2csv

Documentación base del proyecto `json2csv`.

El paquete convierte archivos JSON (arreglo de objetos o JSON Lines) a CSV,
separando la lógica de conversión (`core`) de la interfaz de línea de comandos
(`cli`). La documentación de la API se genera automáticamente con `pdoc` a
partir de los docstrings, en `docs/site/`.
